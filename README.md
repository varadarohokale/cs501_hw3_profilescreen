# The ProfileScreen App
This app implements a profile screen header using Jetpack Compose. The layout demonstrates proper Box layering, including a background header, a circular avatar in the foreground, and a profile card that partially overlaps the header to create a classic “profile card” UI pattern.
This project demonstrates layout control in Compose using the `Box` composable to create layered UI elements. The screen replicates a modern profile header design pattern commonly seen in social media applications.
